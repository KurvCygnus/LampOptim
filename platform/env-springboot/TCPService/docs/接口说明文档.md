## 路灯控制
| 接口名称 | 请求地址 | 请求方式 | 参数格式 | 权限要求 |
|---------|---------|---------|---------|---------|
| 开灯 | /light/open | POST | List<String> ids | 无 |
| 关灯 | /light/close | POST | List<String> ids | 无 |
| 路灯查询 | /light/select | POST | QueryParameter | 无 |
| 添加路灯 | /light/addDevice | POST | List<String> ids | 无 |
| 修改路灯 | /light/update | POST | List<Device> devices | 无 |
| 删除路灯 | /light/delete | POST | List<String> ids | 无 |
| 开启自动感应 | /light/openAuto | POST | List<String> ids | 无 |
| 关闭自动感应 | /light/closeAuto | POST | List<String> ids | 无 |
| 设为故障 | /light/setBroken | POST | List<String> ids | 无 |
| 设为正常 | /light/setNormal | POST | List<String> ids | 无 |
| 全开控制 | /light/openAll | POST | 无 | 无 |
| 全关控制 | /light/closeAll | POST | 无 | 无 |
